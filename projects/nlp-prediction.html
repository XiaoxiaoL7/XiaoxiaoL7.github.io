Key findings

  • Sentiment polarity was the single strongest predictor — negative news articles led actual performance changes by 1–2 weeks
  • Industry-specific named entities (product launches, regulatory mentions) added 8pp to model R²
  • LASSO regularisation identified a sparse 23-feature model nearly as accurate as the full 120-feature model
  • Combined NLP + financial ratio features outperformed financial ratios alone by 22pp in R²
