Key findings from process mining

  • 33% of cases deviated from the happy path — mainly due to incomplete input data triggering manual review
  • Manual approval step added an average 2.1 days to throughput time — the primary bottleneck
  • Workload analysis showed Monday and Tuesday had 40% higher case volumes — resource allocation was unbalanced
  • Recommended: automated pre-validation check before submission to reduce rework loops by an estimated 60%
